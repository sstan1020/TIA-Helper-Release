# TIA Helper

A small floating toolbar that connects to Siemens TIA Portal (via Openness) and lets you
import/compile PLC program blocks with one click — or let an AI assistant do it through
the same interface.

## Download

Grab the latest `TiaWpf.exe` from the [Releases](../../releases) page and run it — no
installer needed. TIA Portal must be installed on the same machine.

## License

Free to use. Right-click the floating icon and choose **License code...** to get your
hardware code, then follow the instructions to request a license.

## Source

This repository hosts release builds only. Source code is maintained privately.
